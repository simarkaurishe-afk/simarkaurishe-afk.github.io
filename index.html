Featured Project

PropConnect

Project Overview

PropConnect is a full-stack web application developed to simplify the process of buying, selling and renting residential and commercial properties.

Sellers can add complete property information and upload multiple images. Buyers and tenants can browse listings, search by city, property type, listing type and budget, and view detailed property information.

Angular TypeScript ASP.NET Core C# SQL Server Entity Framework Core

Core Capabilities

Project Features

User Authentication

Users can register and log in as buyers or sellers. Passwords are securely stored using BCrypt hashing.

Property Management

Sellers can add, update, view and delete property listings through complete CRUD functionality.

Multiple Image Upload

Sellers can upload multiple property images, which are displayed on the property details page.

Search and Filters

Users can search properties using city, budget, property type and listing type filters.

Responsive Interface

The Angular frontend includes landing, login, registration, home, sell, rent and details pages.

REST API Integration

The Angular frontend communicates with ASP.NET Core Web API using HTTP requests and REST principles.
